NotesBlazor - додаток розроблений за допомогою .Net6 з використанням таких технологій як: EntityFramework,PostagreSql,Blazor.
 Проект використовую багатолеєрну архітектуру задля можливості його розширення та кращого тестування кожного леєра.
 Проект містить 4 леєри такі як (Domain,UnitTestProject,Application,Blazor) Кожен з цих леєрів виконує певну функцію у додатку.
 Domain - Леєр у якому зберігається головна модель проекту "Note" леєр зберігає основні класи які працюють з базою даних (Repository та Context). Головною метою цього леєра є забезпечення з'єднання з базою даних за допомогою Repository та зберігання головної модеі "Note"
Application - Леєр у якому знаходяться сервіси, які за допомогою Repository Використовують інформацію з БД та виконують основні функції (
Додавання, Видалення, Редагування, та отримання об'єктів з бази даних)
Blazor - леєр,який забезпечує інтерфейс для користувача він виконує головну візуальну роль додатку, завдяки ньому звичайний користувач може створювати нотатки та використовувати весь доступний  функціонал.
UnitTestProject - зберігає логіку того чи правильно функціонують функції додатку, головною його метою є перевірка правильності працювання функцій які знаходяться у Repository
